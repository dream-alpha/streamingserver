# streamingserver

## Purpose

- This app implements a streaming server for the DreamOS StreamingCockpit plugin. 
- It runs on the dreambox Ubuntu subsystem or on native or WSL Ubuntu.

## Installation

The streamingserver app will be installed as dependency of the StreamingCockpit plugin.
if you want to manually install it on an Ubuntu PC or WSL:
- pip install -i https://test.pypi.org/simple/ streamingserver
